CHANNELS

          
Where OpenClaw can message you (and receive from you). Enable a channel that's already set up. Cron reminders and agent messages get delivered here. To add a new channel, click below - it opens OpenClaw's own setup where you enter the bot token (the cockpit never handles your tokens).

          

          

        

        

          
LOCAL MODEL

          
The QVAC model behind the serve. Switching restarts it (up to a minute) and is remembered across restarts. For the agent to reliably use tools (cron, files, messaging), pick 9B or larger - models under ~7B often fail tool calls (that's why a 4B can't set up a cron from chat).
